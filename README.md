# GENAI

## Overview

GENAI is a repository created to document and organize learning, experimentation, and implementation in the fields of Generative Artificial Intelligence, Machine Learning, and Neural Networks. The repository contains notebooks, notes, scripts, and practical exercises developed while studying and exploring various AI concepts.

This repository serves as a personal learning space as well as a reference for students and enthusiasts interested in AI and related technologies.

## Repository Structure

```text
GENAI/
│
├── .github/workflows/    # GitHub Actions workflows
├── ML/                   # Machine Learning notebooks and implementations
├── NN/                   # Neural Network experiments and models
├── Notes/                # Study notes and reference materials
├── scripts/              # Utility scripts and supporting code
├── requirements.txt      # Project dependencies
└── README.md             # Project documentation
```

## Getting Started

### Prerequisites

Ensure that Python 3.x is installed on your system.

### Clone the Repository

```bash
git clone https://github.com/srtanya-25/GENAI.git
```

### Navigate to the Project Directory

```bash
cd GENAI
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

## Usage

You can explore the repository in any of the following ways:

1. Open the notebooks using Jupyter Notebook or Google Colab.
2. Run Python scripts available in the repository.
3. Use the notes for learning and revision.
4. Modify and extend the implementations for experimentation and research.

To start Jupyter Notebook locally:

```bash
jupyter notebook
```
